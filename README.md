# Drug-Dose-Validator

A Python-based pharmaceutical safety validation system that checks medication doses against predefined limits using custom exceptions, exception handling, and logging.

## Features

* Validates drug dose values.
* Detects invalid data types.
* Prevents negative or zero doses.
* Checks doses against maximum recommended limits.
* Logs validation results and errors for audit purposes.

## Concepts Used

* Classes & Objects
* Custom Exceptions
* Exception Handling (`try/except`)
* Logging
* Pharmaceutical Dose Validation

## Files

* `Drug_Dose_Validator.ipynb` – Jupyter Notebook
* `Drug_Dose_Validator.py` – Python Script
* `dose_error.txt` – Error Log File

## Learning Outcome

This project demonstrates how medication dose validation can be automated using Python while applying patient safety principles, custom exceptions, error handling, and audit trail generation.
